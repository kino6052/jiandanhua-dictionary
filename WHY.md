Why jiandanhua?

The Problem with Current Methods
Comprehensible input videos, graded readers, and HSK‑based main problem is that there is no consistency and creating illusion that this is real Chinese. If we claim from the start that this is a separate language we manage expectations - it is a separate language that is easy to learn and is going to allow your chinese progress skyrocket.

The result from conventional methods: inconsistency, false confidence, and a slow realization that “simple Chinese” is a vague promise, not a real language.

What 简单话 Does Differently
简单话 is the first explicit bridge language to Mandarin.

Explicit vocabulary has a boundary - it is built making sure you learn the most necessary.

Explicit definition system — every word outside the set is defined using only the set.

Explicit layers — Core → Minimal → Extended. A learner knows exactly where they stand.

This is not “simplified Chinese” in the vague sense. It is a formal subset with closure.

With jiandanhua the point is not to master vocab for some level - but to start reading and watching from the get go. 
Because vocabulary is concise it is guaranteed you will encounter words many times.
There are no graded materials for jiandanhua - all our library materials are done in this complete language, that is the point.
Because language has to be used and used like language, not just pretending scenarios.

Because this language is easy to learn - once you cover the lessons - you should be able to watch the materials (using tools of course)

Why That Matters for the Learner
With 简单话, a learner can finally:

Stay inside the bridge — and know they are inside.

Define any unknown word immediately — using only what they already know.

Move to full Mandarin transparently — by replacing definitions with atomic words (看病的人 → 医生).

Measure real progress — from Core to Minimal to Extended, not by guesswork.

No more: “I kind of know HSK 4, but sometimes I don’t.”
No more: “The video said ‘simple’ but I heard 10 new words.”

The Final Sentence
简单话 takes down the walls of inconsistency and illusion. You can finally stay — without falling into the gap between “simple” and “real” — because the bridge is complete under your feet.

The necessity of this language is because Chinese language is really hard for the reason of having too many layers - we stripped it to the necessity methodologically and we are proud of it.
