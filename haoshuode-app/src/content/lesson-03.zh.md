Hǎo-shuō-de 没有格变化——名词不会因为"是动作的发出者"还是"是动作的承受者"而变换词尾。这类信息只住在一个地方：词序。Hǎo-shuō-de 的每一句话都遵循同一个固定形状：

主语 + 动词 + 宾语

换一下顺序，你得到的不是同一句话的另一种语气，而是另一句话——很可能是错的一句。词序不是文体上的选择，它就是语法本身。

你能造出的最简单的句子完全没有宾语——只有主语、动词 <audio-example zh="是">shì</audio-example>（"是"），以及一个用来描述或归类主语的名词：

主语 + <audio-example zh="是">shì</audio-example> + 名词

这个"动词"从不变形。不管是谁在说话，不管在描述什么，<audio-example zh="是">shì</audio-example> 永远是 <audio-example zh="是">shì</audio-example>。

```examples
Zhè-ge shì rén. | 这是一个人。 | | 这个是人。
Zhè-ge shì shuǐguǒ. | 这是一个水果。 | | 这个是水果。
Yī-ge xiě-de dōngxi shì dōngxi. | 写的东西（书/文件）是一个东西。 | | 一个写的东西是东西。
Rén shì nǚrén. | 那个人是女人。 | | 人是女人。
Dòngwù shì dōngxi. | 动物是东西。 | | 动物是东西。
Nǚrén shì rén. | 女人是人。 | | 女人是人。
```

仔细看上面两个例句：<audio-example zh="人是女人">Rén shì nǚrén</audio-example>（"那个人是女人"）和 <audio-example zh="女人是人">Nǚrén shì rén</audio-example>（"女人是人"）。同样三个词，同样的结构，范围却正好相反——一句说的是某一个具体的人，另一句是关于整个类别的通泛判断。句子本身没有任何标记来区分这两种读法，也不需要。Hǎo-shuō-de 的名词本来就不分单复数、不带冠词——回想第二课，数量由上下文决定。这里也是同样的原则：是指某一个具体的东西，还是指整个类别，同样由上下文决定。词本身不变，变的只是你已经知道的情境。

而当你确实需要把数量说清楚时，这个任务依然不会落在名词身上——它落在句子里另一个独立的词上。<audio-example zh="这多的东西是写的东西">Zhè duō-de dōngxi shì xiě-de dōngxi.</audio-example>（"这些[很多]东西是写的东西"，也就是"这些是书"）用了 <audio-example zh="多">duō</audio-example>("多")，通过 `-de` 连接，用法跟形容词一模一样。注意这里少了什么：<audio-example zh="个">-ge</audio-example>。回想第二课，量词的全部职责就是圈定一个可数的单位——<audio-example zh="这个">zhè-ge</audio-example> 精确指向一个东西。而 <audio-example zh="多">duō</audio-example> 根本不是在指向某一个东西，它说的是一整片数量，没有什么可供量词去圈定的了，于是 <audio-example zh="个">-ge</audio-example> 就自然消失了。换句话说，数量从来都不是名词本身携带的属性——它是句子里别的词做出的决定，一个词一个词地决定，取决于那个词在句子里干的是什么活。
