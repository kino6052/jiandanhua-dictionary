```vocab
shuǐ | вода, жидкость | | 水
dìfāng | место (как в пространстве, так и в переносном смысле — часть чего-либо) | | 地方
xiǎo | маленький, небольшой | | 小
hǎo | хороший, простой, дружелюбный | | 好
dà | большой, важный, высокий (о росте) | | 大
```

Прилагательное может напрямую определять существительное, следуя стандартной структуре сказуемого:

`Подлежащее + hěn + Прилагательное`

Когда прилагательное в роли определения непосредственно модифицирует существительное как единое целое, используется структурная частица `-de`.

```examples
Shuǐ hěn hǎo. | Вода хорошая. | | 水很好。
Zhè-ge shì hěn-xiǎo-de dìfāng. | Это маленькое место (комната/помещение). | | 这个是很小的地方。
Wǒ kàn yī-ge hěn-dà-de dòngwù. | Я вижу очень большое животное. | | 我看一个很大的动物。
```
