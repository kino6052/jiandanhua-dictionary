```vocab
shuǐ | вода, жидкость | | 水
dìfāng | место (как в пространстве, так и в переносном смысле — часть чего-либо) | | 地方
xiǎo | маленький, небольшой | | 小
hǎo | хороший, простой, дружелюбный | | 好
dà | большой, важный, высокий (о росте) | | 大
```

Урок 3 дал вам <audio-example zh="是">shì</audio-example> — слово, которое связывает подлежащее с существительным (<audio-example zh="人是女人">Rén shì nǚrén</audio-example> — «человек — женщина»). Прилагательным тоже нужен связующий элемент, но мандаринский не использует для этого <audio-example zh="是">shì</audio-example> повторно — вместо этого используется другое слово: <audio-example zh="很">hěn</audio-example>.

Подлежащее + <audio-example zh="很">hěn</audio-example> + Прилагательное

Скажите <audio-example zh="水好">shuǐ hǎo</audio-example> само по себе — и это не прозвучит как простое, законченное высказывание: оно звучит как половина сравнения, будто что-то осталось несказанным. <audio-example zh="很">hěn</audio-example> — это то, что завершает предложение. В этой позиции <audio-example zh="很">hěn</audio-example> формально означает «очень», но его настоящая работа здесь — просто быть связкой: не стоит читать «очень» в каждое предложение, где оно встречается.

Когда прилагательное стоит прямо рядом с существительным, а не образует отдельное высказывание, — описывая его как единое целое, так же как <audio-example zh="很小的地方">hěn-xiǎo-de dìfāng</audio-example> описывает «маленькое место», — оно присоединяется через `-de`, ту же связующую частицу из Урока 2.

```examples
Shuǐ hěn hǎo. | Вода хорошая. | | 水很好。
Zhè-ge shì hěn-xiǎo-de dìfāng. | Это маленькое место. | | 这个是很小的地方。
Zhè-ge shì yī-ge hěn-dà-de dòngwù. | Это очень большое животное. | | 这个是一个很大的动物。
```

```exercise
Место маленькое.
Это хорошее животное.
Это большой фрукт.
```

```answers
Dìfāng hěn xiǎo. | | 地方很小。
Zhè-ge shì hǎo-de dòngwù. | | 这个是好的动物。
Zhè-ge shì yī-ge hěn-dà-de shuǐguǒ. | | 这个是一个很大的水果。
```
