```vocab
shuǐ | 水，液体 | | 水
dìfāng | 地方，场所（也指某事物的部分） | | 地方
xiǎo | 小的，少量的 | | 小
hǎo | 好的，简单的，友好的 | | 好
dà | 大的，重要的，高大的 | | 大
```

第三课教了你 <audio-example zh="是">shì</audio-example>，它是把主语和名词连接起来的词（<audio-example zh="人是女人">Rén shì nǚrén</audio-example>——"那个人是女人"）。形容词也需要一个连接词，但普通话不会再用 <audio-example zh="是">shì</audio-example> 来干这个活——它用的是另一个词：<audio-example zh="很">hěn</audio-example>。

主语 + <audio-example zh="很">hěn</audio-example> + 形容词

单独说 <audio-example zh="水好">shuǐ hǎo</audio-example>，听起来并不像一句完整、中性的陈述——它更像半句比较，仿佛还有一半话没说完。<audio-example zh="很">hěn</audio-example> 的作用就是把这句话补完整。在这个位置上，<audio-example zh="很">hěn</audio-example> 字面意思是"很"，但它真正的工作只是当连接词——不要把每一句带 <audio-example zh="很">hěn</audio-example> 的句子都理解成"非常"。

当一个形容词直接贴在名词旁边，而不是单独成句——把名词描述成一个整体，就像 <audio-example zh="很小的地方">hěn-xiǎo-de dìfāng</audio-example> 描述"一个小地方"那样——它就要靠 `-de` 连接上去，也就是第二课里那个连接用的助词。

```examples
Shuǐ hěn hǎo. | 水很好。 | | 水很好。
Zhè-ge shì hěn-xiǎo-de dìfāng. | 这是一个小地方。 | | 这个是很小的地方。
Zhè-ge shì yī-ge hěn-dà-de dòngwù. | 这是一个很大的动物。 | | 这个是一个很大的动物。
```

```exercise
这个地方很小。
这是一个好动物。
这是一个大水果。
```
