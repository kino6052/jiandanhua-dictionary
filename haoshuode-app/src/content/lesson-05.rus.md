```vocab
wǒ | я, меня, мы | | 我
nánrén | мужчина, мужского пола | | 男人
nǐ | ты, вы | | 你
qún | сообщество, группа | | 群
xīn | новый, другой, свежий | | 新
```

Местоимения Hǎo-shuō-de ведут себя как любое другое существительное, которое вы уже знаете — <audio-example zh="我">wǒ</audio-example> («я») и <audio-example zh="你">nǐ</audio-example> («ты») просто занимают место Подлежащего из Урока 3, без какого-либо особого обращения.

И, как и любое существительное Hǎo-shuō-de, они не несут собственного числа. <audio-example zh="我">wǒ</audio-example> означает «я» или «мы» в зависимости от того, что уже ясно из ситуации — вспомните Урок 2: ничто в самом слове не отмечает разницу, это делает только контекст вокруг него.

Чтобы показать принадлежность, присоедините местоимение к существительному через `-de` — ту же связующую частицу из Уроков 2 и 4: <audio-example zh="我的">wǒ-de</audio-example> («мой»), <audio-example zh="你的">nǐ-de</audio-example> («твой»). Тот же дефис, та же работа — склеить одно слово с другим в единую описательную единицу — независимо от того, что именно описывает: прилагательное, превращённый в существительное глагол или, как теперь, местоимение.

```examples
Wǒ shì rén. | Я человек. / Мы люди. | | 我是人。
Nǐ shì hǎo-de rén. | Ты хороший человек. | | 你是好的人。
Zhè-ge shì wǒ-de xiě-de dōngxi. | Это мой документ. | | 这个是我的写的东西。
Nǐ-de dìfāng hěn xīn. | Твоё место новое. | | 你的地方很新。
Wǒ-de qún hěn dà. | Моё сообщество большое. | | 我的群很大。
Nánrén-de dòngwù hěn xiǎo. | Животное этого мужчины маленькое. | | 男人的动物很小。
```

```exercise
Твой фрукт хороший.
Это новое сообщество.
Я хороший человек.
```

```answers
Nǐ-de shuǐguǒ hěn hǎo. | | 你的水果很好。
Zhè-ge shì xīn-de qún. | | 这个是新的群。
Wǒ shì hǎo-de rén. | | 我是好的人。
```
