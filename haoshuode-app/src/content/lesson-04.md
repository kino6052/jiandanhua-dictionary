```vocab
shuǐ | water, liquid | | 水
dìfāng | a place (both in space or metaphorical to mean part of something) | | 地方
xiǎo | little, small | | 小
hǎo | good, simple, friendly | | 好
dà | big, important, tall | | 大
```

Lesson 3 gave you <audio-example zh="是">shì</audio-example>, the word that connects a subject to a noun (<audio-example zh="人是女人">Rén shì nǚrén</audio-example> — "the person is a woman"). Adjectives need a connector too, but Mandarin doesn't reuse <audio-example zh="是">shì</audio-example> for the job — it uses a different word instead: <audio-example zh="很">hěn</audio-example>.

Subject + <audio-example zh="很">hěn</audio-example> + Adjective

Say <audio-example zh="水好">shuǐ hǎo</audio-example> on its own, and it doesn't read as a plain, complete statement — it sounds like half of a comparison, as though something's left unsaid. <audio-example zh="很">hěn</audio-example> is what finishes the sentence. In this spot <audio-example zh="很">hěn</audio-example> technically means "very," but its real job here is just to be the connector — don't read "very" into every sentence that uses it.

When an adjective sits right next to a noun instead of making its own statement — describing it as a single unit, the way <audio-example zh="很小的地方">hěn-xiǎo-de dìfāng</audio-example> describes "a small place" — it binds on with `-de`, the same connecting particle from Lesson 2.

```examples
Shuǐ hěn hǎo. | Water is good. | | 水很好。
Zhè-ge shì hěn-xiǎo-de dìfāng. | This is a small place. | | 这个是很小的地方。
Zhè-ge shì yī-ge hěn-dà-de dòngwù. | This is a very big animal. | | 这个是一个很大的动物。
```

```exercise
The place is small.
This is a good animal.
This is a big fruit.
```

```answers
Dìfāng hěn xiǎo. | | 地方很小。
Zhè-ge shì hǎo-de dòngwù. | | 这个是好的动物。
Zhè-ge shì yī-ge hěn-dà-de shuǐguǒ. | | 这个是一个很大的水果。
```
