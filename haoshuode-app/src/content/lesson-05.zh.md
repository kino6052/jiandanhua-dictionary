```vocab
wǒ | 我，我们 | | 我
nánrén | 男人，男性 | | 男人
nǐ | 你，你们 | | 你
qún | 群体，社区 | | 群
xīn | 新的，另一，新鲜的 | | 新
```

Hǎo-shuō-de 的代词表现得跟你已经认识的其他名词一模一样——<audio-example zh="我">wǒ</audio-example>（"我"）和 <audio-example zh="你">nǐ</audio-example>（"你"）直接站到第三课里"主语"的位置上，不需要任何特殊处理。

而且，跟 Hǎo-shuō-de 的每一个名词一样，它们本身不带数量。<audio-example zh="我">wǒ</audio-example> 是"我"还是"我们"，取决于情境本身已经说明的东西——回想第二课：词本身不标记这种区别，只有周围的语境才能说明。

要表示所属关系，把代词用 `-de` 连接到一个名词上——也就是第二课和第四课里那个连接用的助词：<audio-example zh="我的">wǒ-de</audio-example>（"我的"）、<audio-example zh="你的">nǐ-de</audio-example>（"你的"）。同一个连字符，同一种工作——把一个词粘到另一个词上，组成一个单一的描述单位——不管在描述的是形容词、由动词变来的名词，还是现在的代词。

```examples
Wǒ shì rén. | 我是人。 | | 我是人。
Nǐ shì hǎo-de rén. | 你是好人。 | | 你是好的人。
Zhè-ge shì wǒ-de xiě-de dōngxi. | 这是我的文件。 | | 这个是我的写的东西。
Nǐ-de dìfāng hěn xīn. | 你的地方很新。 | | 你的地方很新。
Wǒ-de qún hěn dà. | 我的群体很大。 | | 我的群很大。
Nánrén-de dòngwù hěn xiǎo. | 那个男人的动物很小。 | | 男人的动物很小。
```

```exercise
你的水果很好。
这是一个新的群体。
我是一个好人。
```
